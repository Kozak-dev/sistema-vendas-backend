# 📌 Backend API em C#

Este é um projeto de backend desenvolvido em C# utilizando .NET, com foco em organização de código, criação de APIs e boas práticas de desenvolvimento.

---

## 🚀 Tecnologias utilizadas

* C#
* .NET (ASP.NET Core)
* Visual Studio
* Git e GitHub

---

## 📂 Estrutura do projeto

* **Controllers** → Responsável pelas rotas da API
* **Models** → Representação dos dados
* **Services** → Regras de negócio
* **Data** → Acesso a dados (se houver banco)

---

## ⚙️ Como executar o projeto

1. Clone o repositório:

```bash
git clone URL_DO_SEU_REPOSITORIO
```

2. Abra o projeto no Visual Studio

3. Execute o projeto (F5)

---

## 📡 Funcionalidades

* Criação de endpoints
* Organização em camadas
* Estrutura pronta para integração com banco de dados

---

## 🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar desenvolvimento backend, versionamento com Git e organização de código.
